## Story Template

Copy this when creating a new story:

```markdown
# US-NNN: Short Title

## Story

**As a** [role]
**I want** [capability]
**So that** [benefit]

## Context

[Why does this matter? What problem does it solve? What happens without it?]

## Acceptance Criteria

### AC1: [Name]
**Given** [precondition]
**When** [action]
**Then** [expected result]

### AC2: [Name]
...

## Business Rules

- BR1: [Rule]
- BR2: [Rule]

## Definition of Done

- [ ] All acceptance criteria pass
- [ ] Tests written and passing
- [ ] Documentation updated
- [ ] CHANGELOG updated
- [ ] Code reviewed and merged
```
